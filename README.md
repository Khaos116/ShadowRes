# ShadowRes

This repository keeps only release metadata/history.

Runtime assets are published to GitHub Releases:

- Debug: <https://github.com/Khaos116/ShadowRes/releases/tag/shadowres-debug>

The Android host reads `plugin_manifest.json` and `host_update.json` from the debug release assets.
Use `LiveWos2027/scripts/publish-shadowres.sh` to publish APK/json assets.
